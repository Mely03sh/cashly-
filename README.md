# Cashly - Aplicación de finanzas personales

Aplicación web simple (HTML/CSS/JS) para registrar ingresos y gastos, ver estadísticas visuales, recibir alertas de presupuesto y exportar datos.

## Funciones incluidas

- Registro de ingresos y gastos con categorías.
- Balance general en tiempo real.
- Gráfica de gastos por categoría.
- Estadísticas rápidas (total ingresos, gastos y ahorro).
- Presupuesto mensual configurable con alerta visual al superar el límite.
- Exportación de movimientos a CSV (compatible con Excel).
- Exportación a PDF mediante impresión del navegador.

## Uso rápido

1. Abre `index.html` en tu navegador o ejecuta un servidor local.
2. Define tu presupuesto mensual.
3. Registra ingresos y gastos.
4. Revisa tus métricas y gráfica.
5. Exporta tus datos cuando lo necesites.

## Servidor local opcional

```bash
python -m http.server 8000
```

Después visita: `http://localhost:8000`
